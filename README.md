🍽️ Zomato Restaurant Data Analysis

📌 Project Overview

The objective of this project is to analyze restaurant trends, customer preferences, service availability, and overall performance to generate meaningful business insights.

🎯 Objectives

• Analyze restaurant distribution across countries and cities

• Identify popular cuisines

• Evaluate online delivery and table booking availability

• Analyze restaurant opening trends over the years

• Examine customer ratings and overall satisfaction

• Build interactive dashboards for data-driven insights

🛠️ Tools & Technologies Used

1.	Microsoft Excel – Data Cleaning, Data Modeling, Pivot Tables
   
2.	MySQL – Data Querying, Joins, Aggregations
   
3.	Power BI – Data Modeling, DAX, Interactive Dashboard
  
4.	Tableau – Data Visualization & Dashboard Creation
   
📂 Project Workflow

🧹 Data Cleaning (Excel)

• Removed duplicates and null values

• Standardized formats

• Created calculated columns

🗂️ Data Modeling

• Built relationships between tables

• Created a Calendar Table using DateKey_Opening

• Added Year, Month, Quarter, YearMonth, Weekday columns

🛢️ SQL Analysis (MySQL)

• Used SELECT, JOIN, GROUP BY

• Extracted insights on cuisine, ratings, delivery options

📊 Data Visualization

• Designed interactive dashboards in Power BI and Tableau

• KPI cards for Ratings, Restaurant Count, Delivery Availability

📊 Key Insights

🇮🇳 India dominates the restaurant count, contributing the highest number of restaurants in the dataset.

🚫 The majority of restaurants do not offer online delivery or table booking, indicating potential service expansion opportunities.


🍛 North Indian cuisine is the most popular, followed by other regional cuisines.

📈 Restaurant openings have been consistent over the years, with noticeable peaks in 2011 and 2018.

⭐ The average rating is 2.89, which is relatively low and highlights significant room for improving customer satisfaction and service quality.

🎯 Outcome

• Developed interactive dashboards

• Generated actionable insights

• Supported data-driven business decisions
